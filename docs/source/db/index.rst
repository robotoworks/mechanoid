Mechanoid DB
============
|mechdb| is a code generator for generating sqlite backed content 
providers driven by a simple language (DSL) supporting most of the Sqlite 
language itself and some extra language constructs for configuration and 
managing schema migrations (incremental schema changes).

.. warning:: You must install the Mechanoid Plugin before you can use |mechdb|
